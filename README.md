# HD2 Signed Distance Field Texture Converter
Uses signed distance fields for optimizing texture size without loss in texture quality.

OpenGL implementation of Chris Green's paper : 
__Improved alpha-tested magnification for vector textures and special effects__.

Link to paper : https://dl.acm.org/citation.cfm?id=1281665

Forked from https://github.com/Lilyel/ValveSignedDistanceField
